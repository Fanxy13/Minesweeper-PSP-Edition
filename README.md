# PSP Minesweeper

A simple Minesweeper clone built for the Sony PSP as a homebrew project.

## Features
- Classic grid-based gameplay  
- Random mine generation  
- Flagging system  
- Win/Loss detection  

## Installation
Copy the game folder to `/PSP/GAME/` on your PSP, or run the EBOOT.PBP file in PPSSPP.
